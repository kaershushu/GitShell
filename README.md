# Git Shell:和Git相关的Shell脚本集合:

* 1.查询当前分支绑定的源分支

* 2.设置只能推送到远端同名分支的`pre-push`脚本

## 查询源分支

* 脚本自定义设置

  > 1.设置本地仓库路径。

  > 2.设置本地仓库别名。

* 脚本使用步骤(Mac平台)

  > 1.拷贝脚本文件到`Home目录(~/Home)`下。

  > 2.在`~/.zshrc`里(我使用的是on my zsh)设置执行脚本别名，我设置的是```alias src="sh ~/s.sh"```

  > 3.打开`Terminal`直接输入设置的别名`src`即可
